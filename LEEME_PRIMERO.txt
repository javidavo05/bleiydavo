🆘 ¿PROBLEMAS?

► No puedo iniciar sesión
  → Verifica que creaste los usuarios en Firebase Authentication
  → Email exacto: jvallejo@timeline.com y bvega@timeline.com

► Página en blanco
  → Abre consola (F12) y revisa errores
  → Verifica firebase-config.js

► No se suben fotos
  → Verifica Storage rules publicadas
  → Inicia sesión como jvallejo (admin)

► Los datos no se guardan
  → Verifica Firestore rules publicadas
  → Revisa consola (F12) para errores
